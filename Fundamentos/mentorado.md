# Requisitos de Entrada dos Alunos por Módulo  
Curso Técnico em Tecnologia da Informação

## Diretriz Geral
Os requisitos de entrada dos módulos são baseados nas **competências e conhecimentos adquiridos nos módulos anteriores**, não sendo exigida formação prévia ou experiência profissional externa. O aluno deve demonstrar capacidade de acompanhar os conteúdos teóricos e realizar atividades autônomas semanais.

---

## Módulo 1 – Fundamentos da Computação

### Requisitos de Entrada
- Não há requisitos técnicos prévios
- Alfabetização básica
- Capacidade de leitura e interpretação de textos
- Acesso a computador com internet

---

## Módulo 2 – Sistemas Operacionais e Linux

### Requisitos de Entrada
- Compreensão básica do funcionamento de computadores
- Conhecimento introdutório sobre hardware e sistemas operacionais
- Capacidade de acompanhar explicações técnicas conceituais

---

## Módulo 3 – Terminal e Ferramentas de Linha de Comando

### Requisitos de Entrada
- Conhecimento básico de Linux
- Compreensão da estrutura de diretórios
- Noções iniciais de permissões de arquivos
- Familiaridade básica com ambiente de sistema operacional

---

## Módulo 4 – Controle de Versão com Git

### Requisitos de Entrada
- Capacidade de utilizar terminal para navegação básica
- Noções de manipulação de arquivos e diretórios
- Compreensão básica de como projetos de software são organizados

---

## Módulo 5 – Lógica de Programação e Algoritmos

### Requisitos de Entrada
- Familiaridade com uso de computador e terminal
- Capacidade de seguir raciocínio lógico sequencial
- Noções básicas de versionamento com Git

---

## Módulo 6 – Estruturas de Dados

### Requisitos de Entrada
- Compreensão de lógica de programação
- Capacidade de ler e interpretar código simples
- Conhecimento de variáveis, condicionais, loops e funções

---

## Módulo 7 – Bancos de Dados

### Requisitos de Entrada
- Compreensão de lógica de programação
- Capacidade de estruturar dados de forma lógica
- Noções básicas de como aplicações armazenam informações

---

## Módulo 8 – Programação Orientada a Objetos

### Requisitos de Entrada
- Conhecimento de lógica de programação
- Familiaridade com estruturas de dados básicas
- Capacidade de organizar código em módulos ou arquivos

---

## Módulo 9 – Arquitetura de Software e Serviços

### Requisitos de Entrada
- Compreensão de aplicações e seus componentes
- Noções de orientação a objetos
- Entendimento básico de servidores e redes
- Familiaridade com bancos de dados e persistência

---

## Módulo 10 – Integração de Sistemas e APIs

### Requisitos de Entrada
- Compreensão de arquitetura de aplicações
- Conhecimento básico de serviços e comunicação entre sistemas
- Noções de APIs e requisições
- Capacidade de acompanhar fluxos de integração

---

## Módulo 11 – Projeto Final (TCC)

### Requisitos de Entrada
- Conclusão satisfatória dos módulos anteriores
- Capacidade de planejar e executar uma solução técnica
- Familiaridade com versionamento de código
- Compreensão de arquitetura, banco de dados e integrações
- Autonomia para desenvolvimento e documentação de projeto
